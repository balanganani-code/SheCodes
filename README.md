This is a responsive frontend web application built using HTML, CSS, and JavaScript.

The application allows users to:

Switch between two different themes

Book a private session via an interactive popup form

Receive a personalized confirmation message after entering your name and email

This project was built as part of my frontend development journey while learning through SheCodes.

FEATURES OF THE PROJECT
Theme toggle button(light/dark mode)
Responsive Design
A personalised message showing confirmation
